# UFO
Crack the codeword to stop aliens abducting your friend.

<img width="631" height="414" alt="image" src="https://github.com/user-attachments/assets/591083ef-38cc-4a06-aa08-d8fad54cefe3" />

## How to Play
Launch from the terminal and try to guess a codeword chosen randomly from a list of 500 possibilities. Or set the game up for a friend by adding your own codeword as a command-line argument.
